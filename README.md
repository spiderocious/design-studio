# Studio

A portfolio of complete, opinionated design systems, each built as its own
worldview: one stance, one accent, one voice, real scenes instead of component
grids.

**Live:** _(add your Netlify URL here once deployed)_

Each project on the gallery is a full system — foundations (palette, type,
geometry, motion), primitives, data-display, and at least three fully-designed
scenes that prove the system in a real situation.

---

## Running locally

It's static HTML. No build step.

```bash
open index.html
# or serve it: python3 -m http.server 8000  ->  http://localhost:8000
```

## How this is published

This repo is generated from a private design playground. The gallery and every
project under `projects/` are the public artifacts; the design process (briefs,
notes, agent tooling) stays private. Content here is build output — edit it in
the source playground and re-publish, don't hand-edit `projects/`.
